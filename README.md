##课程总结
首先非常非常感谢老师和助教老师这一学期的付出，虽然很遗憾因为这学期选课比较多，同时还在实习，平常并没能抽出很多时间去练习学习，但是仍然收获到了很多。对于考试情况，因为在考试前有多个其他考试以及期末作业并行，因此在机考前我把整个机考复习重心放在了前面较简单的题上，并决定将整体复习重心主要放在笔试方面。因为训练不足并没有顺利作出机考题，我也在后续笔试前更加完整认真的进行了复习，熟读了两遍书并完整梳理了练习册中的所有知识。最后虽然本人真的学的非常不好，但是真的觉得收获到了很多，明确了该如何学习并练习数算，在寒假期间我会根据老师所给的资料继续学习练习，争取可以掌握课程中的所有知识。谢谢老师！
